# Fleets Component

The Fleets component displays a series of workshop items with images and descriptions.

## Installation

Ensure you have Node.js installed. Install the necessary packages using npm:

```bash
npm install next/image react

## Usage

Import the component in your project:

import Fleets from './Fleets';

Use the Fleets component in your JSX:

<Fleets />


## Styling Notes

This component uses Tailwind CSS classes for styling.

.ok a:hover:after provides a hover effect with a white underline.
.texteone:hover changes text color on hover.
.image-wrapper .date and .image-wrapper .slide-indicators provide styling for date and slide indicators respectively.
.workshop-list a::before adds a blue bullet point before workshop links.
Feel free to adjust styles and content as per your project requirements.
